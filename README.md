# EstudianteUniversidad

Este proyecto fue desarrollado en C# como parte de la Unidad 1 de la asignatura de PProgramación Net 1

# Funcionalidades
- Captura el nombre completo del estudiante.
- Captura el año de nacimiento.
- Captura la carrera.
- Captura el promedio general.
- Captura el número de materias inscritas.
- Calcula la edad del estudiante.
- Calcula las horas de estudio semanales (4 horas por materia).
- Reorganiza el nombre en el formato:
  - Apellido paterno
  - Apellido materno
  - Nombre
- Muestra toda la información en pantalla.

# Tecnologías utilizadas

- C#
- .NET 10
- Visual Studio Code
- Git
- GitHub
